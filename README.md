## Description

A Angular PoC to integrate with Zabbix Webhooks

![image](https://user-images.githubusercontent.com/1216181/104812854-6bb46f80-5805-11eb-8adf-85287f45691a.png)


## Dependencies

- **Microsoft signalR**: to implement websockets with the backend
- **ng-event-bus**: to implement pub/sub inside Angular
- **primeNG**: frontend framework
- **ang-jsoneditor**: angular JSON editor

## Install Zabbix UI Dependencies:
```
npm install
```

## Start Zabbix UI:
```
ng serve
```

## Production Zabbix UI compilation:
```
ng build --prod
```
